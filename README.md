# 环境要求

- Python 3.6+
- Node 10+

# 使用


## 代码启动

从代码启动，如下命令：
```
$ cd codepy-server
$ pip3 install -r requriements.txt
$ python3 ./src/main.py

$ cd codepy-web
$ yarn
$ yarn start
```
